# Colibrí OS · PMV Reputation Lab & Market

Este repositorio contiene el **PMV estático** (HTML/CSS/Bootstrap/JS) del ecosistema Colibrí OS:

- **Colibrí Reputation Lab (Emprendedor)**  
  Panel de vuelo que muestra la evolución de un NFT Colibrí según las microacciones y evidencias educativas.

- **Colibrí Reputation Lab (Mecenas / Aliado Semilla)**  
  Panel de impacto que permite visualizar el portafolio de Colibrís apoyados y las becas activadas.

- **Colibrí Reputation Market**  
  Comparador de startups que utiliza las 7 categorías troncales del Modelo Educativo Colibrí LATAM.
